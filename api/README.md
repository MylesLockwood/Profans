## Description

API for xfans project.

## Installation 

### use node 13 (using node version manager - nvm use 13)

```bash
$ npm install 
```

### add mongo URI to .env

### run redis server (sudo service redis-server start)

## Running the app

```bash
# watch mode
$ npm start:dev
```

### Migration

```bash
$ node migrate.js
```
